# Projeto ZERO DISPERDICIO

Protótipo web focado em reduzir desperdício — interface em HTML/CSS/JavaScript e estrutura básica de servidor em Node.js.

## Tecnologias

- JavaScript
- HTML
- CSS
- Node.js
- npm

## Visão geral

Esse repositório contém um protótipo chamado "Projeto ZERO DISPERDÍCIO" com uma interface frontend (index.html, script.js, style.css) e arquivos iniciais de backend (app.js) e pastas preparadas para rotas, modelos e middlewares. O objetivo é servir como base para uma aplicação que ajude a minimizar o desperdício por meio de funcionalidades de compartilhamento, registro e gerenciamento de recursos.

## Funcionalidades (visão geral)

- Interface interativa em index.html com scripts em script.js
- Estilização em style.css
- Servidor básico em app.js (Node.js)
- Estrutura para rotas, modelos e middlewares (pastas vazias ou preparadas)

## Estrutura do repositório

- .gitignore
- app.js — servidor / ponto de entrada (Node.js)
- index.html — página principal
- script.js — lógica do frontend
- style.css — estilos
- package.json / package-lock.json — dependências e scripts
- middLewares/ — middlewares do servidor
- models/ — modelos de dados
- routes/ — rotas do servidor

## Requisitos

- Node.js (versão LTS recomendada)
- npm

## Instalação

1. Clone o repositório:

   git clone https://github.com/Digo1214/Projeto-ZERO-DISPERDICIO.git
2. Entre na pasta do projeto:

   cd Projeto-ZERO-DISPERDICIO
3. Instale as dependências:

   npm install
4. Inicie a aplicação:

   - Se houver script no package.json: `npm run start`
   - Ou execute diretamente: `node app.js`

5. Abra o navegador em http://localhost:<porta> (verifique app.js para a porta utilizada)

## Uso
